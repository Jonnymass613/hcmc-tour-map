# Publish this site with GitHub Pages (no Git required)

This folder (`hcmc-tour-map/`) is a complete static site.

## 1) Create a new GitHub repo
- Go to https://github.com/new
- Repo name: `hcmc-tour-map` (or any name you want)
- Visibility: **Public**
- Click **Create repository**

## 2) Upload the files
- In the new repo, click **Add file → Upload files**
- Drag and drop ALL files from this folder:
  - `index.html`
  - `app.js`
  - `styles.css`
  - `README.md`
  - `UPLOAD_INSTRUCTIONS.md` (optional)
- Click **Commit changes**

## 3) Enable GitHub Pages
- Repo → **Settings** → **Pages**
- **Source**: Deploy from a branch
- **Branch**: `main`
- **Folder**: `/ (root)`
- Save

## 4) Your public URL
- Your site will be available at:
  - `https://<YOUR_GITHUB_USERNAME>.github.io/<REPO_NAME>/`

It may take 1–5 minutes to go live the first time.
